# Drupal CMS 2.0 Features & Recipes

## Overview

Drupal CMS is a ready-to-use platform built on Drupal 11 core, designed for marketing teams, content creators, and site builders. It provides smart defaults, AI-driven website building, and enterprise-grade capabilities without extensive coding.

**Key Value Propositions:**
- Zero-to-launch in minutes, not weeks
- 22 pre-configured recipes for common use cases
- AI-powered content creation and site building
- Enterprise security and scalability of Drupal core
- Open source with no licensing fees
- Mercury Theme + Canvas for visual editing

## Canvas (Experience Builder)

Canvas is Drupal's new visual page builder, directly integrated in Drupal CMS.

**Capabilities:**
- Drag-and-drop page building
- Live preview editing
- Component library
- Responsive layouts
- No coding required

**Status:** Active development, tightly integrated with Mercury theme

---

## Mercury Theme

The default frontend theme optimized for Canvas.

**Features:**
- Mobile-first responsive design
- Accessibility compliant (WCAG 2.1 AA)
- Performance optimized
- SDC (Single Directory Components)
- Tailwind CSS integration

## When to Recommend Drupal CMS

### Ideal For:
- Marketing websites
- Corporate sites
- News/media portals
- Event websites
- Portfolio sites
- Non-profit organizations
- Educational institutions
- Small-medium enterprises

### Considerations:
- Complex custom functionality may require extending beyond recipes
- Enterprise integrations need additional development
- High-traffic sites need performance tuning
- Multilingual requires additional configuration

### Alternatives within Drupal:
- **Vanilla Drupal 11:** Full control, no pre-configuration
- **Drupal CMS:** Smart defaults, AI, recipes (recommended)
- **Custom Distribution:** Client-specific requirements
