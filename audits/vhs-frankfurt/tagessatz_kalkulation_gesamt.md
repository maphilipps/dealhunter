# Tagessatz-Kalkulation Gesamt-Übersicht
## VHS Frankfurt - Alle Positionen mit optimiertem Tagessatz

**Projekt:** VHS Frankfurt Webinfrastruktur
**Datum:** 2026-01-08
**Methodik:** Realistischer Aufwand (adessoCMS + KI) × Tagessatz / Ausschreibung = Optimierter TS

---

## Berechnungsformel

```
Optimierter_TS = (Realistischer_Aufwand_PT × gewählter_TS) / Ausschreibung_PT
```

**Beispiel Position 1.1:**
- Ausschreibung: 6 PT
- Realistisch: 4 PT
- Gewählter TS: €900

```
Optimierter_TS = (4 × 900) / 6 = €600
```

**Interpretation:** Wir arbeiten zum effektiven Preis von €600/Tag und liefern trotzdem das volle Budget-Volumen.

✅ **Marktgerecht:** €900/Tag = Standard für Senior Drupal-Entwicklung + KI-Tools
